76=head2 fastafmt
77
78The term "fastafmt" is a generalization of multifasta format sequence
79files. A fastafmt file only assumes that records start with ">" as the
80first byte of the stream or directly following a new line.
81
82
83

86 size_t countRecords( std::istream& iStream );

98 );

101 const size_t& numToPrint,
102 /***/ std::ostream& oStream = std::cout

104
105These functions read records from I<iStream> and print them to
106I<oStream>. They stop after reading I<numToPrint> records (if given)
107or after encountering an EOF. The assume iStream currently points to
108start of record or EOF. and return the number of records output.
109
110
111=head2 Argument order and defualt values
112
113For functions that input streams or files that argument comes first. This gives
114the functions a semi-object oriented feel. Output stream reference arguments are
115given a default value of std::cout where possible, but input reference arguments
116are not given a default value.
