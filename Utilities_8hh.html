Detailed Description

Utility functions for NGS sequence processing and DNA manipulation.

This header provides core utility functions for:

  • File path operations
  • DNA sequence conversions (ACGT <-> numeric)
  • Vector and map printing utilities
  • Template-based generic printing functions
Note
IMPORTANT: This is a header file. Never use 'using namespace std;' in headers! It pollutes the namespace of all files that include this header. All standard library types use explicit std:: prefix.
