8 * Description: Square Bit Matrix where the element (i, j) is always the same as (j, i).
9 *
10 * Requirements: space: requires ceiling n(n-1)/16 bytes of storage for an n x n matrix.
11 *
12 * Usage: A SymmetricBitMatrix object can be constructed with a starting size or as an empty matrix.
13 * An empty matrix can be made into a matrix with a given size with setSize( size_t n ).
14 * setSize() may only be called for an empty matrix.
15 *
16 * The operator () is overridden to access elements. SymmetricBitMatrix( i, j ) returns the bit at i, j,
17 * or equivalently the bit at j, i.
18 *
19 * Bits can be set or cleared with set(), and clear() methods.
20 * Normally indices are checked for validity, i.e. i, j < n. This small overhead can be avoided by using the
21 * methods clearFast, and setFast.
22 *
23 * Extensions: The current implementation does not provide space for diagonal elements, (i, i). An extention of this
24 * class could easily be added to do that, and in fact virtual declarations are made to facilitate
25 * that extension. This would require ceiling n(n+1)/16 bytes of storing.
26 */
