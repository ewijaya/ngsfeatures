9 * Purpose: Provide an efficient bit matrix.
10 *
11 * Usage: The operator () is overridden to access elements. BitMatrix( i, j ) returns the bit at i, j
12 * If just one argument is given BitMatrix( j ) returns the bit at the jth position of the current
13 * column. Thus BitMatrix.setCurCol( i ); BitMatrix( j ) returns the same bit as BitMatrix( i, j ).
14 *
15 * Normally the indices i, j in these examples are checked for overflow. However there are two ways
16 * to avoid this small overhead. One is to use the methods clearFast, and setFast.
17 *
18 * The more complete way to avoid this overhead however is to use the derived class BitMatrixFast. This
19 * class does not check indices for overflow unless (for debugging) the macro
20 * BITMATRIX_ALWAYS_CHECK_INDICES is defined.
21 *
22 * Initial Value: After constructing a non-empty BitMatrix or using setSize(), all elements of the matrix are
23 * initialized to false.
24 *
25 */
