14 * Purpose: Provide a fixed length array class for fixed length elements that checks for the
15 * validity of indices.
16 *
17 * Description: The FLArray family of classes can correctly hold fixed length elements.
18 *
19 * As the name suggests MallocArray uses malloc, rather than new T[size()];
20 * to allocate memory.
21 *
22 * This is C-like and produce slightly faster code at the cost of having to be
23 * careful not to read uninitialized elements. One convenience it provides is the
24 * ability to hold classes that do not provide a null constructor. See usage.
25 *
26 * MallocArray is *not* the correct choice for variable length type elements or
27 * pointer containing types such as std::string in which the pointed to material
28 * is also considered as part of the object.
29 * It also does not guarantee calling constructors or
30 * destructors on the contained elements. The FLArray family of classes is provided
31 * for variable length types such as std::string
32 *
33 * MallocArray is a fixed length array that checks the size_t index for being
34 * too large when using the operator [].
35 *
36 * MallocArrayFast is a derived class that avoids that small overhead by not checking
37 * the index passed to operator [] for being too large. However, when debugging
38 * MALLOCARRAY_ALWAYS_CHECK_INDICES can be defined to force index checking with
39 * MallocArrayFast too.
40 *
41 * MallocNumArray is an extension of MallocArray which assumes elemType can do arithmetic.
42 * It supplies methods such as sum() which returns the summation of the elements.
43 *
44 * MallocNumArrayFast is exactly analogous to MallocArrayFast. In fact I tried to have it
45 * multiple inherit from MallocNumArray and MallocArrayFast -- but I couldn't get that to
46 * work...
47 *
48 * Usage: MallocArray can be used like this:
49 *
50 * class StructLikeClass{
51 * public:
52 * StructLikeClass( double x, double y ) : x(x), y(y){}
53 * double getY() const{ return y; }
54 * private:
55 * double x;
56 * double y;
57 * };
58 * ...
59 *
60 * cbrc::MallocNumArray<StructLikeClass> structs( 2 );
61 * structs[0] = StructLikeClass( 0.6, 7.8 );
62 * std::cout << "y value of first struct is: " << structs[0].getY() << std::endl;
63 *
64 * Note that StructLikeClass does not have a null constructor!
65 */
