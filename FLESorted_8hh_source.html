125=head1 SYNOPSIS
126
127B<FLESorted>E<lt>I<T>E<gt> is a compact datastructure which maps objects
128of type I<T> to serial numbers and vice versa.
129
130
131
132=head1 DESCRIPTION
133
134B<FLESorted>E<lt>I<T>E<gt> maps a set of n unique objects of type I<T> to the
135serial numbers 0..n-1, where the serial number of each object corresponds
136to its rank when the objects are sorted by C<std::sort>.
137
138I<T> should be an element type suitable to be held in B<FLEArray>, i.e.
139I<T> should be a self contained fixed length chunck of memory which
140can be correctly copied with C<memcpy>. Most builtins such as C<int>, C<double>
141work fine as do simple non-pointer-containing structs. C<std::string> does not
142qualify because it is variable length.
143
144
