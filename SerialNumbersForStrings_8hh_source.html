192=head1 SYNOPSIS
193
194Class for mapping strings to serial numbers
195
196=head1 DESCRIPTION
197
198B<SerialNumbersForStrings><serialNumT> is a template class for mapping
199strings onto serial numbers. I<serialNumT> is the unsigned integer type
200(such as size_t) used to hold serial numbers.
201
202=head1 Serial Numbers
203
204For this class serial numbers a series of successive integers
205starting with zero and ending with one minus the number of strings
206contained in the object.
207
208=head1 Implementation Notes
209
210=head2 Time/Space Efficiency
211
212This container is implemented as a wrapper around a vector of strings.
213The structure is very compact and allows strings to be looked up by
214serial number in constant time. Unfortunately linear time is required
215to look up serial numbers by their strings.
216
217=head1 Constructors
218
219In addition to a null and copy constructor, a B<SerialNumbersForStrings>
220object can be constructed from a vector of std::strings.
221
222 SerialNumbersForStrings( const std::vector<std::string>& strings );
223
224=head1 Methods with generate new serial numbers
225
226Some methods may generate new serial numbers:
227
228 serialNumT getSerialNumber( const std::string& s );
229
230void registerNewSerialNumber ( const std::string& s );
231void registerNewSerialNumbers( const std::vector<std::string>& s );
232
233B<getSerialNumber> is the most basic method. If string I<s>
234is contained in the B<SerialNumbersForStrings> object its
235serial number is returned. Otherwise a new serial number is
236issued and returned.
237
238B<registerNewSerialNumber>(s) assume the string I<s> is known
239to be new to the object. When compiled with NDEBUG, it may be
240significantly faster than B<getSerialNumber>.
241
242=head1 Accessors
243
244 size_t size() const;
245
246B<size()> returns the number of strings contained in the object.
247

249

264=head2 Access by serial number
