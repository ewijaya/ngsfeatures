9 * Description: Bijection from residueIndex strings to successive integers
10 * from 0 to sigma^k - 1, for string of length k < maxStringLen() where
11 * maxStringLen() is determined by sigma and the size of
12 * ResidueIndexMap::indexT.
13 *
14 * The mapping is defined by interpreting the length k
15 * residueIndex string as a base sigma length k integer.
16 *
17 * The most significant base sigma digit (sigit) is
18 * defined to be the first character in the
19 * string. Equivalently, the least significant sigit is
20 * the last character in the string.
21 *
22 * Note this class maps a ResidueIndex string onto a single
23 * integer. This is different from ResidueIndexMap which maps a
24 * string of ResidueIndex onto a string of integers.
25 *
26 * Purpose: Created for use with motif discovery programs.
27 *
28 */
